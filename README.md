# Easy Chat

AI chatbot application powered by Google Gemini API.

## Live Demo
https://easychat-plum.vercel.app/

## Features
- AI-powered conversations using Google Gemini
- Real-time responses
- Clean, responsive UI

## Tech Stack
- React.js
- Tailwind CSS
- Google Gemini API

## Related Repositories
- Backend: https://github.com/vivek7941/EasyChat-backend

## Setup

Install dependencies:
```bash
npm install
```

Run:
```bash
npm start
```
